# 🏋️ FITNESS CLUB — Gym & Fitness Plan Generator

A web application that generates personalized workout and diet plans based on the user's BMI, fitness goal, and lifestyle inputs.

---

## 🚀 Features

- Calculates **BMI** from user inputs (weight, height, age, stress level)
- Classifies users into fitness categories: Underweight, Normal, Overweight, Obese
- Generates a **personalized exercise plan** based on BMI category
- Provides a **custom diet recommendation** for each category
- Clean and responsive web interface

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Python, Django |
| Database | SQLite3 |
| Frontend | HTML, CSS |
| Server | Django Development Server |

---

## 📸 How It Works

1. User enters their **weight, height, age, stress level, and fitness goal**
2. App calculates **BMI** automatically
3. Based on BMI, the app shows:
   - Your BMI score and category
   - A **3-exercise workout plan** with reference links
   - A **diet plan** suited to your category

---

## ⚙️ How to Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/vaishnvich45-dotcom/FITNESS-CLUB.git

# 2. Navigate to project folder
cd FITNESS-CLUB/gymproject

# 3. Install Django
pip install django

# 4. Run the server
python manage.py runserver

# 5. Open in browser
http://127.0.0.1:8000
```
